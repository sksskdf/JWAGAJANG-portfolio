## 개발스택
- JSP, Java Servlet API, jQuery, JavaScript, MySQL, Apache Tomcat

## 서비스 기능
- 메인 페이지(각 주제 별 상품리스트 간단히 모아 보기)
- 회원가입 / 로그인 / 장바구니 / 관리자 페이지 / 마이페이지 
- 공지사항 / 공지 글 쓰기, 수정 / 공지 글 보기
- 문의사항 / 문의 글 쓰기, 수정 / 문의 글 보기
- 카테고리(드롭다운) / 상품목록 / 상세페이지
